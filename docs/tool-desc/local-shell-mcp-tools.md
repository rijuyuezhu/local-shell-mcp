# local-shell-mcp built-in tool surface

This document reflects the current built-in MCP tools registered from `src/local_shell_mcp/tools/registry` on the `improve-tool-descriptions-drop-git` branch.

## Summary

- Built-in MCP tools: **47**
- Git porcelain tools: **0**. Agents should use `run_shell_tool` or `remote_run_shell_tool` for git workflows.
- `audit_tail`: removed. Audit logging still exists internally, but there is no model-facing audit-tail tool.
- `secret_scan`: retained as a bounded heuristic pre-commit/pre-release scan for common credential patterns.
- `apply_patch`: retained as a file-editing tool that applies unified diffs with `git apply`; it is not a git workflow tool.
